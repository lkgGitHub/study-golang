
# 设计模式六大原则
1. 单一指责原则（Single Responsibility Principle）
2. 依赖倒置原则（LSP liskov substitution principle）
3. 接口隔离原则 
4. 里氏替换原则
5. 迪米特原则（law of demeter LOD）高内聚低耦合
6. 开闭原则

1. 简单工厂模式（Simple Factory）
2. 工厂方法模式（Factory Method）
3. 抽象工厂模式（Abstract Factory）
4. 创建者模式（Builder）
5. 原型模式（Prototype）
6. 单例模式（Singleton）
